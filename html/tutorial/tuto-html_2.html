Exemplos de tags HTML

Todo documento HTML deve começar com a determinação do seu tipo <!DOCTYPE html>.

Após começa o HTML de fato com a tag <html> e termina com a tag </html>.

A parte visivel do documento HTML na página fica entre <body> e </body>. 

<!DOCTYPE html>
<html>
	<body>
		Minha primeira página HTML!
	</body>
</html>

Cabeçalhos

Os cabelhaços (heading) HTML são definidos pelas tags <h1> à <h6> , sendo o tamanho da fonte menor quanto maior o h.

<!DOCTYPE html>
<html>
	<body>
		<h1>Este é um cabeçalho H1!</h1>
		<h2>Este é um cabeçalho H2!</h2>
		<h3>Este é um cabeçalho H3!</h3>
		<h4>Este é um cabeçalho H4!</h4>
		<h5>Este é um cabeçalho H5!</h5>
		<h6>Este é um cabeçalho H6!</h6>
	</body>
</html>

Parágrafos

Os parágrafos em HTML são definidos pela tag <p>. 

<!DOCTYPE html>
<html>
	<body>
		<p>Este é um parágrafo!</p>
		<p>Este é o segundo parágrafo!</p>
	</body>
</html>
